主要功能：
JournalGenerator 类提供以下功能：
加载头寸数据 - 支持CSV和Excel文件
计算头寸调整 - 对比当前和前期头寸，计算变动
生成日记账分录 - 自动生成借方/贷方分录
导出到Excel - 包含多个工作表（日记账、按证券汇总、按账户汇总）
导出到CSV - 便于进一步处理
使用方法：
# 直接运行示例
python auto1.py

# 或在你自己的代码中使用
from auto1 import JournalGenerator

generator = JournalGenerator(output_dir='output')
positions = generator.load_position_data('your_positions.csv')
adjustments = generator.calculate_adjustments(positions)
journal_entries = generator.generate_journal_entries(adjustments)
generator.export_to_excel(journal_entries)
数据格式要求：
你的头寸数据文件应包含以下列：
security_id - 证券代码
account - 账户
quantity - 数量
value - 金额
脚本会自动在 output/ 目录下生成日记账文件，包含Excel和CSV两种格式。
